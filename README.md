# Diluter

A utility tool for calculating cell culture dilutions for plating onto petri dishes or others. It calculates the dilution required to achieve a target number of cells/colonies/CFUs within a 100µL volume typically used for plating based on a hemacytometer count.

## Versions

### Diluter 2.0 (Web)

A web application that runs in the browser.

**Usage:**
1.  Open `Diluter2/index.html`.
2.  Enter the average cell count (from a 1/25 square) and the desired number of colonies.
3.  Click "Calculate" to get the dilution protocol.

### Diluter 1.0 (Python)

The original command-line script.

**Usage:**
1.  Run `python Legacy/Diluter1.0.py` in your terminal.
2.  Follow the prompts to enter the cell count and desired colony number.
